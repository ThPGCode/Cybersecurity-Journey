# Day-02 -- Linux file & Directory commands

## Objectives:
- Learn basic Linux navigation.
- Understand File & Directory management.

## Commands learned:

### 1. pwd
Print Working Directory (current location).

bash:
pwd

### 2. ls
List of the files and directories

bash:
ls = Normal.
ls -l = detailed listing.
ls -a = all the files including hidden files.
ls -la = detailed listing + hidden files.
ls ~ = lists of the contents of home directory.

### 3. cd
Change Directory ( one folder to another jump)

bash:
cd [name of the file / folder]
cd .. = moving upward of the path.
cd ~ = jump to the starting point / home directory.

### 4. mkdir
Creating new folder

bash:
mkdir [name of the folder you want to make]

### 5. rm 
removing 

bash:
rm [file name]
rmdir [directory name]

### 6. cp 
copy

bash:
cp [name of the file have to be coppied] [name of the file/folder where have to copy]
